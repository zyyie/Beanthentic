Frequently Asked Questions

To reset your password, go to the Account Security section and fill out the password reset form. You'll need to enter your current password and then provide a new password that meets our security requirements (minimum 8 characters).

Two-Factor Authentication adds an extra layer of security to your account. After enabling 2FA, you'll need to enter a code from your authenticator app (like Google Authenticator) in addition to your password when logging in.

1. Go to Account Security section
2. Click "Enable 2FA"
3. Scan the QR code with your authenticator app
4. Save the backup codes in a safe place
5. Enter the verification code to complete setup

Backup codes are one-time use codes that you can use to access your account if you lose your authenticator device. Each code can only be used once. Keep them in a secure location.

Your activity log is available in the Activity Log section. It shows all recent actions performed on your account, including login times, password changes, and security settings updates. You can search through the log using the search bar.

Go to the Notifications section where you can toggle email, SMS, and in-app notifications for different types of events like system updates, user registrations, and security alerts. Click "Save Notification Settings" to apply your changes.

Yes, we take security seriously. All passwords are encrypted, we support 2FA, and we maintain detailed activity logs. We recommend enabling 2FA and using a strong, unique password for maximum security.
